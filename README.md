# Student Data Management System (C Language)

A console-based application developed in C to manage student records using
struct, file handling, and modular programming.

## Features
- Add student data
- Delete student data
- Display student list
- Sort data alphabetically
- Persistent storage using text files

## Concepts Used
- Struct & Array
- File I/O (fopen, fread, fwrite)
- Modular programming
- Basic data validation

```bash
gcc Project\ Data\ Mahasiswa.c -o mahasiswa
./mahasiswa
